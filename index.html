WRIST‑MOUNTED PUNCH TEMPO MONITOR

A wearable system that detects punch events from wrist motion and computes real-time punch tempo (PPS) on-device.

The ESP32 processes motion data locally and transmits performance metrics wirelessly via BLE to the web dashboard.
